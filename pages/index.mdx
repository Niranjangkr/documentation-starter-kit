# Agile Receivables API

Welcome to the Agile Receivables API documentation.

Use these docs to:

- Sync **customers** and **accounts (invoices)** from your billing systems.
- Enroll accounts into **campaigns** for automated collections.
- Record **payments** and track outstanding balances.
- Read and respond to **disputes** raised by customers.

## Getting started

- Start with the [Accounts &amp; API Keys Overview](/accounts/intro) to understand how workspaces,
  organizations, and API keys interact with accounts and other resources.
- Read [Core Concepts](/accounts/base-docs) for authentication, response envelopes, errors,
  pagination, and rate limits.
- Dive into the [API Reference](/accounts/apis/customers) for endpoint-level details.

## Related apps

These docs accompany the Agile Receivables frontend and backend services. For UI-driven configuration
of API keys and workspaces, use the main web application.
